# موقع مقهى (مشروع تدريبي)

## نبذة عن المشروع
هذا المشروع عبارة عن موقع إلكتروني لمقهى تم تصميمه وتطويره **لغرض التعلم والتدريب على تطوير واجهات الويب**.  
الهدف من المشروع هو تطبيق ما تعلمته في تقنيات الويب وبناء واجهة مستخدم حديثة.

## تنبيه
هذا الموقع **ليس موقعًا رسميًا لأي مقهى أو شركة**، ولم يتم إنشاؤه بالتعاون مع أي جهة.  
تم إنشاء هذا المشروع **كمحاكاة وتجربة تعليمية فقط** بهدف تطوير المهارات في تصميم وتطوير المواقع.

جميع الأسماء أو الصور أو التصاميم المستخدمة هي **لأغراض العرض والتجربة فقط**.

## التقنيات المستخدمة
- HTML
- SCSS
- JavaScript
- Webpack
- Bootstrap

## هدف المشروع
- التدريب على تطوير واجهات المواقع
- تحسين مهارات تصميم واجهة المستخدم (UI)
- تطبيق الأدوات الحديثة في تطوير الويب

## حالة المشروع
مشروع تعليمي وتجريبي فقط وليس للاستخدام التجاري.
